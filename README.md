## üõ† For Developers

If you want to modify or build this app yourself:

### Prerequisites
*   Node.js (v18+)
*   NPM

### Setup
```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/pomo-desktop.git

# Go to the app directory
cd desktop-app

# Install dependencies
npm install
```

### Development
Run the app in development mode with live reload:
```bash
npm start
```

### Build (.exe)
Create a standalone Windows executable:
```bash
npm run build
```
This will generate a `dist/` folder containing the executable.
